### Run Locally

To run **Watch** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/adnan-bhaldar/Watch-Website.git
```

Windows:

```bash
git clone https://github.com/adnan-bhaldar/Watch-Website.git
```
